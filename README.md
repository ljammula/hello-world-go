# hello-world-go

A minimal "Hello, World!" Go project generated by the OpenClaw assistant.

## Run

CLI mode (default):

```bash
go run main.go
go run main.go --mode=cli --lang=te
```

HTTP mode:

```bash
go run main.go --mode=http --addr=:8080 --index=index.html
```

In HTTP mode, the server serves `index.html` at `/`.
